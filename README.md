🚀 DriveApp – Cloud File Storage System

DriveApp is a cloud-based file storage application inspired by Google Drive.
It allows authenticated users to upload, download, and delete files securely, with cloud storage handled via Cloudinary.

This project focuses on real-world backend concepts like authentication, file lifecycle management, and cloud storage integration.

✨ Features

🔐 User Authentication

JWT-based login & signup

Secure session handling using cookies

☁️ Cloud File Storage

File uploads using Multer

Files stored securely on Cloudinary

Supports images, PDFs, and other file types

🗂 User-Specific Files

Each user can access only their own files

Protected routes with authorization checks

🗑 File Deletion

Deletes file from Cloudinary

Syncs deletion with MongoDB

🎨 Clean UI

Dark-themed dashboard

Simple and responsive interface

🛠 Tech Stack

Backend

Node.js
